# Screenshots anwenden

Standard-Tastenkombination zum Erstellen eines Screenshots vom gesamten Bildschirm:
```
WINDOWS-TASTE + Druck
```

Standard-Pfad zum gespeicherten Screenshot:
```
%HOMEPATH%\Pictures\Screenshots\
```
